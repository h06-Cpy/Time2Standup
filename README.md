# Time to Stand Up 📐
> 건강한 업무 습관을 위한 당신만의 AI 자세 코치

## 👋 소개
"잠깐 일어서서 스트레칭 하셨나요?"

장시간 앉아 있는 것이 건강에 해롭다는 것은 알지만, 업무에 집중하다 보면 시간을 잊기 쉽습니다.
**Time to Stand Up**은 웹캠을 통해 실시간으로 당신의 자세(앉기/서기)를 인식하고, 시각적인 피드백으로 건강한 균형을 찾아주는 웹 애플리케이션입니다.

## ✨ 주요 기능

### 1. 실시간 자세 인식 (AI Powered)
별도의 웨어러블 기기 없이, 컴퓨터의 웹캠만으로 당신이 앉아 있는지 서 있는지 자동으로 감지합니다.
> 🔒 **프라이버시 보호:** 카메라는 오직 자세 인식용으로만 사용되며, 내 모습은 흐릿하게 처리되거나 숨겨집니다. 영상은 어디에도 저장되거나 전송되지 않습니다.

### 2. 직관적인 상태 알림 'The Orb'
화면 중앙의 신비로운 '오브'가 현재 상태를 알려줍니다.
- **서 있을 때:** 상쾌한 **민트색** 오브가 둥실 떠오릅니다. 🌿
- **앉아 있을 때:** 차분한 **라벤더색** 오브가 바닥에 머뭅니다. 🪑

### 3. 습관 밸런스 게임
하루 동안의 '앉은 시간'과 '서 있는 시간'이 마치 줄다리기(Tug of War)를 하듯 경쟁합니다.
서 있는 시간을 늘려 화면 속의 '앉기' 영역을 밀어내세요! 게임처럼 즐기며 건강을 챙길 수 있습니다.

### 4. 30분 건강 경고
한 번에 너무 오래 앉아 있지 않도록 도와줍니다.
- **30분 이상** 계속 앉아 있으면 화면이 붉게 변하며 경고를 보냅니다. 🚨
- 이때 자리에서 일어나 가볍게 스트레칭을 하면, 경고가 사라지고 다시 건강한 상태로 돌아옵니다.

## 🚀 사용 방법
1. 앱을 실행하고 **카메라 권한**을 허용해주세요.
2. 카메라가 상반신을 비추도록 각도를 조정합니다.
3. 브라우저 탭을 켜두고 평소처럼 업무를 보세요.
4. 붉은 경고가 뜨면 잠시 일어나 몸을 풀어주세요!
