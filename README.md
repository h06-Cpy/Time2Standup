# 📐 Time to Stand Up

## 1. 디자인 철학: "줄다리기 (Tug of War)"
자세 데이터를 단순한 기록이 아닌, **"좋은 습관(서기)"**이 **"나쁜 습관(앉기)"**을 밀어내는 게임처럼 시각화합니다.

## 2. 컬러 시스템 (다크 모드)
- **배경:** `#121212` (딥 차콜) - 눈의 편안함.
- **서기 (활동):** `#00FFC2` (네온 민트) - 긍정적, 에너제틱.
- **앉기 (휴식):** `#7050FF` (딥 라벤더) - 정적, 좌식.
- **경고:** `#FF4B4B` (알림 레드) - **30분** 동안 계속 앉아 있을 때 발동.

## 3. 레이아웃 & 컴포넌트
3단 수직 구조.

### A. 헤더 (상단) - "상태 & 제어"
- **왼쪽:** 로고 텍스트 (예: *Posturize*)
- **오른쪽:** `[● REC]` (점멸) / `[CAM ON/OFF]` 토글.
- **기능:** 카메라는 기본적으로 숨겨지거나 흐리게 처리됩니다. 토글하여 자신의 모습을 확인(모니터링 느낌)할 수 있어 프라이버시 부담을 줄입니다.

### B. 메인 (중앙) - "오브 (The Orb)"
- 중앙에 떠 있는 **오브**.
- **서 있을 때:** 민트색 오브가 떠다니거나 숨쉬는 애니메이션.
- **앉아 있을 때:** 라벤더색 오브가 납작해지거나 바닥에 붙은 애니메이션.
- **역할:** 한눈에 상태 확인.

### C. 푸터 (하단) - "밸런스 게이지" ⭐
- **형태:** 두꺼운 가로 바 (너비 80%).
- **구조:** 왼쪽에 `서기 (민트)`, 오른쪽에 `앉기 (라벤더)`. 영역은 전체 시간의 비율을 나타냅니다.
- **경고 발동:**
  - **30분 이상** 계속 앉아 있으면 앉기 바가 `빨간색`으로 변하고 빛납니다.
- **데이터 표시:**
  - 왼쪽: `서기 00:45:10`
  - 오른쪽: `앉기 01:20:00`
  - 하단 중앙: `총 02:05:10`

## 4. UX 시나리오
1. **초기 상태:** 카메라 권한 요청. 회색 오브 대기.
2. **분석:** 앉아 있으면 라벤더 바가 채워짐. 서 있으면 민트 바가 채워짐.
3. **사전 경고:** 계속 앉아 있는 시간이 30분에 가까워지면(예: 25분), 바 가장자리가 빨간색으로 펄럭임.
4. **경고:** 계속 앉아 있는 지 30분이 되면, 앉기 바가 **밝은 빨간색**으로 변하고 배경이 붉은 틴트로 어두워짐. "MOVE YOUR BODY" 문구 표시.
5. **해결:** 사용자가 일어서서 스트레칭을 함. 민트 바가 빨간색/라벤더 영역을 밀어내어 게이지를 "정화"하고 건강한 균형으로 되돌림.
